# 💻 VPN自動設定スクリプト

## `VPN_Create.bat`

このスクリプトは、Windows環境にVPN接続設定を自動で作成するためのバッチファイルです。

### ✨ 概要と特長

* **手間いらずの自動設定**: VPN名、サーバーアドレス、事前共有鍵（PSK）などの必要な接続情報をPowerShellコマンド（`Add-VpnConnection`）で一括設定します。
* **自動管理者昇格**: 実行時に管理者権限が不足している場合、PowerShellを利用して自動的に**管理者権限でスクリプトを再起動**します。

---

### 🚀 使用方法

1.  `VPN_Create.bat` ファイルをダウンロードします。
2.  ファイルを**ダブルクリックして実行**します。
3.  スクリプトが自動でVPN接続を作成します。

---

### 🌐 接続手順

スクリプト実行後、作成されたVPN接続は以下の方法で利用できます。

#### 1. タスクバーから接続

1.  タスクバーにある**ネットワークアイコン**をクリックします。
2.  新しく追加されたVPN接続を選択し、「接続」を押します。
3.  接続時に求められる**ログインアカウント情報**（ユーザー名とパスワード）を入力します。

#### 2. Windowsの設定から接続

1.  `設定`を開き、**ネットワークとインターネット** > **VPN** へ進みます。
2.  新しく追加されたVPN接続が表示されていることを確認します。
3.  「接続」ボタンを押して、接続時に求められる**ログインアカウント情報**を入力します。
